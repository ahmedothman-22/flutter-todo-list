# 📱 Tadweer - To-Do App (OS2 Project)

**Tadweer** is a creative and efficient To-Do application built with **Flutter**, designed to help you manage your daily tasks with clarity and ease. 

> 💡 **What is "Tadweer"?**
> The name **"Tadweer"** means *"cycle"* or *"rotation"* in Arabic. Since productivity is a continuous loop of planning, executing, and reviewing, **Tadweer** helps you keep that cycle going smoothly.

*This project was developed as a practical course project for the **Operating Systems 2 (OS2)** class, showcasing modern mobile development practices, state management, and cloud integration.*

---

## ✨ Features

* **📝 Complete CRUD Operations:** Add, update, edit, and delete tasks seamlessly.
* **✅ Task Tracking:** Easily mark tasks as completed or pending.
* **🔐 Secure Authentication:** User authentication (sign up/login) powered by **Firebase Authentication**.
* **☁️ Cloud Sync:** Real-time task storage and synchronization using **Cloud Firestore**.
* **🔄 Advanced State Management:** Built using both **Provider** and **Cubit** to demonstrate clean and scalable state handling.
* **🎨 Modern UI:** Clean, responsive, and minimal user interface.

---

## 🛠️ Tech Stack & Architecture

* **Framework & Language:** Flutter & Dart
* **Database & Backend:** 
  * **Cloud Firestore:** For real-time cloud task storage.
  * **Firebase Authentication:** For secure user access.
  * **SharedPreferences:** For lightweight local data persistence.
* **State Management:**
  * **Provider**
  * **Cubit (BLoC)**
